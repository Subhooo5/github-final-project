# github-final-project

## Project Name
**Simple Interest Calculator** (IBM Java Backend Assignment)

## Description
A Bash script that computes simple interest from user-supplied values for principal amount, annual rate of interest, and time period in years.

The formula used is:

```
Simple Interest = (Principal × Rate × Time) / 100
```

## Input Fields
- **Principal** — the initial amount of money
- **Rate of Interest** — the annual interest rate, as a percentage
- **Time Period** — the duration, in years

## Usage

```bash
chmod +x simple-interest.sh
./simple-interest.sh
```

The script prompts for each input and prints the computed simple interest.
